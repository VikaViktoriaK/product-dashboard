# React E-commerce SPA
### React + TypeScript + Vite
Современное одностраничное приложение для просмотра товаров с интегрированным чатом поддержки.

## Технологический стек
* **React 19** + **Vite** (быстрая сборка)
* **TypeScript** (строгая типизация)
* **TanStack Router** (типизированная навигация и роутинг)
* **TanStack Query** (эффективное управление серверным состоянием)
* **Tailwind CSS** (адаптивная и современная верстка)
* **Axios** (клиент для API запросов)
* **WebSockets** (реализация чата в реальном времени)

## Основные возможности
* **Каталог товаров:** Список товаров с пагинацией и фильтрацией по категориям и цене.
* **Умный поиск:** Автодополнение (Autocomplete) с выпадающим списком при вводе названия.
* **Страница товара:** Детальное описание, галерея изображений и блок похожих товаров.
* **Поддержка (Чат):** WebSocket-чат с эхо-сервером, поддерживающий автоскролл и отправку по Enter.
* **Адаптивность:** Полная поддержка мобильных устройств.

##  Запуск проекта
1. Установите зависимости:
npm install
2. Запустите проект в режиме разработки: npm run dev 
3. Для сборки production-версии:
npm run build
